# Golang Maps / Anahtar-Değer İlişkisi

Array ve Slice yapılarında özel bir isteği bulmak için bütün elemanlara bakmamız ve veri tipini belirtmemiz gerekiyordu. Aynı şekilde maps'lerde key-value prensibine dayansada, veri türlerini belirtmemiz gerekiyor.

Map'lerde key-value çifti olduğu için, key olarak string, value olarak int gibi örneklerle kullanabiliriz.

Python kullandıysanız, dictionary yapısıyla karşılaşmışsınızdır.Go'da karşılığı ise Maps'dir.
