HTTP Nedir? 
======
**H**yper **T**ext **T**ransfer **P**rotocol ifadesinin kısaltmasıdır. İstemci ile sunucu arasındaki veri akışının kurallarını belirleyen protokoldur. 
İstek – Cevap (request, response) modeline göre çalışır.
